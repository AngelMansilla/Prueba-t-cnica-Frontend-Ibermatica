# Contexto del Producto: Gestión de Usuarios

## Problema a Resolver
Actualmente no existe una interfaz centralizada y eficiente para gestionar la información de los usuarios (Demandantes y Empleados). La gestión manual puede llevar a errores e inconsistencias.

## Solución Propuesta
Una aplicación web Angular que:
- Proporciona una interfaz de usuario intuitiva para listar, ver, crear y editar usuarios.
- Diferencia la información relevante según el tipo de usuario, mostrando estudios para demandantes y experiencia laboral para empleados.
- Asegura la validación de datos en los formularios para mantener la calidad de la información.

## Experiencia de Usuario Deseada
- **Eficiente:** Los usuarios deben poder realizar las tareas de gestión rápidamente.
- **Clara:** La información debe presentarse de forma organizada y fácil de entender.
- **Consistente:** La interfaz y el comportamiento deben ser coherentes en toda la aplicación.
- **Adaptable:** La aplicación debe funcionar correctamente en diferentes tamaños de pantalla (diseño responsive).
