# Resumen del Proyecto: Gestión de Usuarios

## Objetivo Principal
Construir una aplicación Angular para gestionar usuarios, permitiendo operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y mostrando detalles específicos según el tipo de usuario (DEMANDANTE o EMPLEADO).

## Características Clave
- Listado de usuarios paginado y/o filtrable.
- Vista detallada de cada usuario.
- Formulario de creación y edición de usuarios.
- Secciones dinámicas en el formulario y detalle para mostrar Estudios (si es DEMANDANTE) o Experiencia Laboral (si es EMPLEADO).
- Navegación clara entre las diferentes vistas.
